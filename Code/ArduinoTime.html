The space rover you are about to control consists of an interactive terminal that allows you to control basic movement of the rover.

Control Terminal that will be used

Image of the terminal is shown above.

Up, Down, Left, and Right Buttons in the terminal

The up and down buttons shown allow you to move the rover forward or backward depending on what you would like it to do. To move left and right, use the left and right buttons. To stop the rover from moving, press the stop button in the middle of the terminal.

Color Sensor inside the terminal

The color sensor button allows you to use the integrated color sensor that you will be using for the activity for these rovers.
